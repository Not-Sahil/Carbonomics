Lab Fleet Monitor

Aggregate carbon tracking across all connected devices

🖧 Run Agent on Each Lab Machine
Install the lightweight agent on each device in your lab/company. It auto-registers and sends heartbeats every 30s.
# 1. Install dependencies
pip install psutil requests
 
# 2. Run agent (replace YOUR_SERVER_IP)
python device_agent/agent.py \
  --server http://YOUR_SERVER_IP:5000 \
  --region ap-south-1 \
  --device-type laptop
 
# Device auto-registers and heartbeats every 30s
